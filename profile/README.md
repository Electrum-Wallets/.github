## Key Features of Electrum Wallets

- **Seed Phrase Backup** – **Electrum Wallets** generates 12/24-word recovery phrases.  
- **Hardware Wallet Integration** – **Electrum Wallets** connects to Ledger, Trezor, KeepKey.  
- **Multi-Signature Support** – Create 2-of-3 or 3-of-5 wallets with **Electrum Wallets**.  
- **Cold Storage Offline Signing** – **Electrum Wallets** never exposes keys online.  
- **Custom Transaction Fees** – Set per-byte fees manually in **Electrum Wallets**.  
- **UTXO Management** – **Electrum Wallets** lets you freeze, label, and select coins.  
- **Replace-By-Fee (RBF)** – Bump unconfirmed transactions via **Electrum Wallets**.  
- **Tor & Proxy Support** – Route traffic anonymously with **Electrum Wallets**.  
- **CLI & JSON-RPC** – Automate Bitcoin tasks using **Electrum Wallets** command line.  
- **Label Synchronization** – Sync wallet labels across devices using **Electrum Wallets**.  
- **Lightweight Protocol** – **Electrum Wallets** does not require full blockchain download.  
- **Electrum Server Compatibility** – Connect to custom servers via **Electrum Wallets**.  
- **Partial Transaction Signing** – Collaborate on transactions safely with **Electrum Wallets**.  
- **BIP32 & BIP39 Support** – Industry-standard key derivation in **Electrum Wallets**.  
- **Transaction History Export** – Export to CSV or JSON from **Electrum Wallets**.
